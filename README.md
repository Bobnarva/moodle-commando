# Moodle Universal Commando

Скрипт для автоматизации и помощи в прохождении тестов на базе Moodle. Работает на основе связки локальной базы данных Firebase и API Gemini.

## Основные функции
* **Быстрая загрузка:** Интерфейс плашки появляется сразу при готовности каркаса страницы, не дожидаясь загрузки тяжелых картинок или формул.
* **Автоматический сбор ответов:** При просмотре результатов теста (страница `review.php`) скрипт автоматически сохраняет правильные ответы в облачную базу.
* **Поддержка ИИ:** Если вопроса нет в базе, его можно решить через нейросеть Gemini (поддерживаются текстовые вопросы, тесты со множественным выбором, сопоставления и вопросы с картинками).
* **Режим автопилота:** Кнопка автоматического последовательного прохождения страниц теста.

## Установка

1. Установите расширение **Tampermonkey** в браузер.
2. Откройте в этом репозитории файл `commando.user.js`.
3. Нажмите кнопку **Raw** в верхнем правом углу панели с кодом.
4. В открывшемся окне Tampermonkey нажмите **Установить** (или *Обновить*).

## Обновление
Обновление происходит автоматически в фоновом режиме через Tampermonkey по мере изменения версии (`@version`) в коде этого репозитория.
